## 🚀 快速开始

### 环境要求

- **Node.js**: >= 16.0.0
- **npm**: >= 7.0.0 或 **yarn**: >= 1.22.0

### 安装步骤

#### 1. 克隆项目
```bash
git clone <your-repository-url>
cd order-app
```

#### 2. 安装依赖
```bash
npm install
# 或
yarn install
```

#### 3. 启动开发服务器
```bash
npm run dev
# 或
yarn dev
```

服务器将在 `http://localhost:5173` 启动

#### 4. 构建生产版本
```bash
npm run build
# 或
yarn build
```

#### 5. 预览生产构建
```bash
npm run serve
# 或
yarn serve
```

#### 背景色
```css
--background: hsl(250 20% 12%)   /* 深蓝紫 #1E1A2E */
--card: hsl(260 20% 18%)         /* 卡片背景 */
--muted: hsl(260 15% 22%)        /* 柔和背景 */
```

#### 文字色
```css
--foreground: hsl(320 30% 95%)   /* 主文字色 */
--muted-foreground: hsl(320 15% 65%) /* 次要文字色 */
```

### 字体系统
- **正文字体**: Noto Sans SC（思源黑体）
- **标题字体**: Zhi Mang Xing（志莽行书）
- **等宽字体**: Menlo

### 圆角系统
- **小圆角**: 12px (`--radius-sm`)
- **中圆角**: 14px (`--radius-md`)
- **大圆角**: 16px (`--radius-lg`)
- **超大圆角**: 20px (`--radius-xl`)

### 动画效果
- **悬浮放大**: 卡片 hover 时放大 1.02 倍

## 🗂️ 数据结构

### 菜品数据（Dish）
```typescript
interface Dish {
  id: string;              // 菜品ID
  name: string;            // 菜品名称
  description: string;     // 菜品描述
  price: number;           // 价格
  image: string;           // 图片URL
  category: string;        // 分类
}
```

### 购物车项（CartItem）
```typescript
interface CartItem extends Dish {
  quantity: number;        // 数量
}
```

### 订单数据（Order）
```typescript
interface SavedOrder {
  id: string;              // 订单ID
  date: string;            // 下单时间（ISO格式）
  items: Array<{           // 订单商品
    name: string;
    quantity: number;
    price: number;
  }>;
  total: number;           // 订单总价
  name: string;            // 顾客姓名
  address: string;         // 配送地址
  notes: string;           // 订单备注
  payMethod: string;       // 支付方式（wechat/alipay）
}
```
