# 🌟 甜蜜小厨 - 项目重要文件分析报告

## 📋 项目概览

**项目名称**: 甜蜜小厨 (Sweet Kitchen)  
**项目类型**: 浪漫主题的在线点餐应用  
**技术栈**: React 18 + TypeScript + Vite + Tailwind CSS 4  
**主题风格**: 浪漫星空主题（粉紫色调 + 星空背景）

### 1. 菜单浏览与点餐

## 📁 重要文件分类与分析

## ⭐ 一级重要文件（核心业务逻辑）

### 1. `src/App.tsx` - 应用根组件
**重要性**: ⭐⭐⭐⭐⭐  
**作用**: 
- 应用的最顶层组件，负责整体架构搭建
- 配置全局 Provider（QueryClient、Tooltip、Cart）
- 设置路由系统（使用 Wouter）
- 集成星空背景和全局 Toast 通知

**关键依赖**:
- `@tanstack/react-query` - 数据请求管理
- `wouter` - 轻量级路由库
- `CartProvider` - 购物车状态管理
- `StarsBackground` - 星空背景效果

**路由配置**:
```
/ → Home (首页菜单)
/cart → Cart (购物车)
/checkout → Checkout (结算页)
/orders → Orders (订单历史)
```

---

### 2. `src/context/CartContext.tsx` - 购物车状态管理
**重要性**: ⭐⭐⭐⭐⭐  
**作用**: 
- 使用 React Context 管理全局购物车状态
- 提供购物车操作方法（添加、删除、更新数量、清空）
- 使用 localStorage 持久化购物车数据
- 自动计算总价和总数量

**核心功能**:
- `addToCart(item)` - 添加菜品到购物车
- `removeFromCart(id)` - 从购物车移除菜品
- `updateQuantity(id, quantity)` - 更新菜品数量
- `clearCart()` - 清空购物车
- `totalItems` - 购物车商品总数
- `totalPrice` - 购物车总价

**数据持久化**: 使用 `localStorage` 保存购物车数据

---

### 3. `src/data/menu.ts` - 菜单数据源
**重要性**: ⭐⭐⭐⭐⭐  
**作用**: 
- 定义完整的菜单数据结构
- 包含 4 个分类：浪漫主菜、甜蜜甜品、温馨饮品、特色小食
- 每个菜品包含：名称、描述、价格、图片、分类

**数据结构**:
```typescript
{
  id: string,
  name: string,
  description: string,
  price: number,
  image: string,
  category: string
}
```

**菜品总数**: 16 道菜品（每个分类 4 道）
